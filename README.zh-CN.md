## ✦ 快速开始

```bash
# 克隆项目
git clone git@github.com:JOYCEQL/simresume.git
cd simresume

# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev
```

打开浏览器访问 `http://localhost:3000` 即可开始制作简历。

## ✦ 构建与部署

```bash
# 生产构建
pnpm build

# 启动生产服务器
pnpm start
```

### Docker

```bash
docker compose up -d
```
